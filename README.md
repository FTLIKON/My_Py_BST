### 二叉树的定义与构造


二叉树的定义是Python与c++代码风格的主要区别  
先看看我的c++二叉树定义：

```c++

struct node
{
    node *l, *r; //当前节点的左右树
    int v;       //节点值
    node(int vv, node *ll, node *rr) { l = ll, r = rr, v = vv; }
};

```

观察可知，主要由两部分构成：
1. 指向自己的指针，也就是左右结点
2. 构造函数

在Python中，构造函数通过定义class中的 “__init__” 函数来实现，而左右节点的指针可以通过“__init__”中的self参数来指向自身的其他参数来实现，代码如下：

```python

class node(object):
	def __init__(self,l,r):
		self.l = l # 当前节点的左右树
		self.r = r 
        self.v = 0 # 节点值

```
